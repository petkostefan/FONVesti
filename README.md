# FONVesti

**Problem:** Chasing a bachelor's degree can be hard, but geting the news for every subject in semester can be even harder! Every subject has its own website, and getting the news for each subject in a semester can be really time consuming.

**Solution:** Flask web application that gets the news from each subject and groups them into semesters, so if you for example want to check the news for subjects in forth semester, you simply press a button and the application scrapes the latest news for each subject from their wabsites, sorts and presents them. News posted in the last five days are highlighted. If you want to read the whole post, click on _Read more_ and you will be redirected to that post's page.
If you only want news for a specific subject, click on it's heading and the application will list you all the latest posts for that subject.

Link to demo: https://fonvesti.herokuapp.com/
